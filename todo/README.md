# React 사용한 App 만들기

- 함수형 컴포넌트로 간단한 모바일 버전의 MyToDoApp 만들기

## :: 목적

- 리액트에 조금 더 익숙해지도록 연습하기
- 개인적인 투두 리스트앱 만들어보기

### :: 구현

- 컴포넌트 세분화
- useState 사용
- usEffect 사용
- {children} 인자로 받아오기
- 반복되는 것을 map을 사용하여 간결하게
